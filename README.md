# jmcc

### entity _interaction_
- needs tag "jmcc"
- needs data.jmcc.left.type with value "attack"
- needs data.jmcc.right.type with value "interaction"
- can have data.jmcc.left.function which will run as/ at the interaction entity on attack
- can have data.jmcc.right.function which will run as/ at the interaction entity on interaction
